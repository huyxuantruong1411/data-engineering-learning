reportRecommended by The_Weekly_Jay
Quite literally Naruto if the series took place in a dystopian future where ninjutsu was replaced with ancient weaponry. It takes so many concepts from Naruto that I wouldn't be surprised if Shueisha had taken some legal action against Enix. This series reads like if someone tried creating a Naruto reboot from memory and started throwing in their OCs as replacements for the original cast. A couple of differences here and there, but if you've read Naruto and end up reading 666 Satan (Stupid name btw, especially for 2001 standards.) then you'll see how shamelessy similiar the two are. Yet, despite it being a shameless clone, its actually enjoyable? I mean it reads like Naruto, it LOOKS like Naruto, yet it isn't. You know how some people say they'd erase their memory to rewatch their favorite show? 666 Satan is perfect for that. If you ever feel nostalgic for some Naruto vibes but don't want to reread 700 chapters, then 666 Satan (or O-Parts Hunter, the objectively better name) is perfect. VERY similiar, yet distinct enough to be its own enjoyable story.  read more
